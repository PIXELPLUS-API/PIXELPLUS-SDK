Detailed Description

Lightweight, dependency-minimal UTF-8/UTF-16 helpers and enum stringifiers for IPM.

  • Windows uses Win32 WideChar APIs (no extra deps).
  • Linux/macOS fallback uses std::wstring_convert (codecvt) for simple tools-layer conversions.

Also provides wchar_t* stringifiers for public IPM enums declared in CIpmEnv.h and CIpmGpuEnv.h (e.g., ipm::En_CpuType, ipm::En_GpuType, ipm::SupportState).
