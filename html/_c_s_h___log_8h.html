Detailed Description

Simple thread-safe logging class with printf-style and complete message logging.

This header defines the CSH_Log class, which provides a singleton logger with configurable log levels, file output, and thread safety.

◆ LogLevel

enum class cshlog::LogLevel : int
strong

Log severity levels from most severe to most verbose.

Lower numeric values indicate higher severity. The value LogLevel::Count is not a real severity; it is a sentinel that indicates the number of levels.

Enumerator
Fatal 

Fatal error that prevents the program from continuing.

Error 

Recoverable error: operation failed but process continues.

Warn 

Suspicious or unexpected condition that may require attention.

Info 

High-level informational message about normal operation.

Debug 

Detailed messages useful when debugging.

Trace 

Very fine-grained messages for deep tracing.

Count 

Sentinel; number of log levels (not to be used as a level).
